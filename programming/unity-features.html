Introduction

It's been awhile since I upgraded Unity. Have been stuck on Unity 5 for a long time supporting older projects and it has taken me awhile to catch up with all the new amazing features that the current versions now ships with.

Below is a comprehensive list of 31 major Unity features released since Unity 5, organized as a reference guide to save you time digging around.
